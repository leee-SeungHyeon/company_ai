# 인사 규정

## 1. 연차 휴가

### 1.1 연차 발생 기준
- 입사 1년 미만: 매월 개근 시 1일 발생 (최대 11일)
- 입사 1년 이상: 15일 기본 발생
- 3년 이상 근속 시 2년마다 1일 추가 (최대 25일)

### 1.2 연차 사용 방법
- 연차는 최소 반일 단위로 사용 가능
- 사용 3일 전까지 팀장 승인 필요
- 연속 5일 이상 사용 시 부서장 승인 필요

### 1.3 연차 소멸
- 미사용 연차는 해당 연도 12월 31일 소멸
- 소멸 전 연차 사용 촉진 공지 발송

---

## 2. 병가

- 연간 최대 60일 사용 가능
- 3일 이상 병가 시 의사 진단서 제출 필요
- 병가 첫날부터 급여 100% 지급

---

## 3. 경조사 휴가

| 구분 | 일수 |
|------|------|
| 본인 결혼 | 5일 |
| 자녀 결혼 | 1일 |
| 부모/배우자 사망 | 5일 |
| 형제자매 사망 | 3일 |
| 조부모 사망 | 2일 |

---

## 4. 출퇴근 및 근무 시간

- 표준 근무 시간: 09:00 ~ 18:00 (점심 1시간)
- 유연근무제 신청 가능: 08:00~10:00 사이 출근, 8시간 근무
- 재택근무: 팀장 승인 하에 주 2회까지 가능
