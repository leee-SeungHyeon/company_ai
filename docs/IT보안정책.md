# IT 보안 정책

## 1. 비밀번호 정책

- 최소 8자 이상, 영문+숫자+특수문자 조합 필수
- 90일마다 비밀번호 변경 의무
- 이전 5개 비밀번호 재사용 금지
- 5회 연속 로그인 실패 시 계정 잠금 (IT팀 문의)

---

## 2. 장비 보안

- 자리 비울 때 반드시 화면 잠금 (Windows: Win+L / Mac: Ctrl+Cmd+Q)
- 회사 노트북 분실 시 즉시 IT팀 신고
- 개인 USB 사용 금지, 회사 지급 USB만 사용 가능
- 무단 소프트웨어 설치 금지

---

## 3. 데이터 보안

- 고객 정보 및 기밀 문서는 사내 시스템에만 저장
- 개인 이메일로 업무 자료 전송 금지
- 클라우드 저장소(개인 Google Drive 등) 업무 자료 저장 금지
- 퇴사 시 모든 업무 자료 반납 및 삭제 확인

---

## 4. 네트워크 보안

- 공용 Wi-Fi 사용 시 VPN 필수 접속
- VPN 계정 타인과 공유 금지
- 의심스러운 이메일 첨부파일 클릭 금지 → IT팀 신고

---

## 5. 위반 시 조치

- 1차 위반: 경고 및 보안 교육 이수
- 2차 위반: 징계위원회 회부
- 고의적 정보 유출: 즉시 해고 및 법적 조치
